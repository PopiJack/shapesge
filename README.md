#Shapesge
Možnosť pridať konfiguračný súbor sbge.ini a v nich nastaviť jednotlivé nastavenia:
1.Okno
  [Window]
  Title = "titul okna"
  ShowInfo = true/false
  FPS = 100
  Width = 1000(v px)
  Height = 700(v px)
  
2.Timer 
  [Timers]
  sekunda = 1000(v ms)
  
3. Myš
  [Mouse] 
  klik = clicked LEFT
  
4. Farby 
  [Colors]
  red = #farbav16sustave
  
5. Klavesnicu
  [Keybord]
  q = pressed q
  ...
